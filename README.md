# ERTriageEnv - Hospital ER Triage OpenEnv Environment

A comprehensive AI agent training environment that simulates hospital emergency room triage operations.

## Project Structure

This project implements a complete ER triage simulation with:
- Patient generation and scenario management
- Triage decision evaluation
- Performance grading and reward systems
- Baseline implementations and testing

## Installation

```bash
cd er-triage-openenv
pip install -r requirements.txt
```
